# Weather App

```
This is a basic weather app that refreshes itself after every 5 mins by default, which can be changed from the config file.
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
